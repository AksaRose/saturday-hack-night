Getting Started

Loading...

To participate in this Hacknight all you have to do is build something cool using GitHub API. Here are some ideas to get your creativity flowing,

  • CV Maker - This is a webpage where people can get their CV generated using data from their github profile. You can stop there or add some crazy features and blow us away!
  • Follow The Tree - Simply put this is a webpage where people come to feel notlonely. Users should be able to generate a tree of their follower by entering in their username.
  • Random Repo - This is a page for those who like to explore. The idea is each time you refresh the page you will be show a list of random repos to explore.

Submitting Your Creations

You all must be used to submitting something after you are done building, but here it is the other way around ( twist... twist ! ).

Here is what to do,

Step 1

Create a repo This shouldn't need any more explanation but still, the team lead should create a repo in which you will be building the project.

Step 2

Create or Join a Team If you are the team lead, you should create a team here and add members to the team. The team members will get an invite to their mail once the team is created.

Step 3

Start Codding Finally! You can start making your dream project. At this point, our team will be monitoring your progress and giving you timely updates. Try to commit code frequently so that we can get back to you in case of any error.
